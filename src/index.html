How to Use HoML

Pull a model from Hugging Face Hub

Download a model to your local machine.

homl pull gemma-3-4b-it

Run a model in interactive chat mode

Start a conversation with a model.

homl run gemma-3-4b-it

You can also use the full Hugging Face model ID:

homl run google/gemma-3-4b-it

Start the OpenAI-compatible API server

Serve models through an API.

homl serve
