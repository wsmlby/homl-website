Download HoML

Get started with HoML on your home server.

One-line Installation

Run the following command in your terminal to install the HoML CLI:

curl -sSL https://homl.dev/install.sh | sh

This script will detect your platform and install the correct version of the HoML CLI.

After installing the CLI, run the following command to install and start the HoML server:

homl install

Manual Installation

If you have already installed the HoML server manually (e.g., using Docker), you can configure the CLI to connect to it.

Run the following command to configure the socket path for the CLI:

homl config --socket <socket-mount-path-host>

Replace <socket-mount-path-host> with the actual path to the mounted socket.
